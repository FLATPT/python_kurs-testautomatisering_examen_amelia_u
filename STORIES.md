# USER STORIES

["Läslistan"](https://tap-ht25-testverktyg.github.io/exam/) är en webbsida där man kan välja ut sina favoriter från en katalog med böcker, eller lägga till nya.  
Den riktar sig till studenter och har som mål att öka studenters läsandet genom att lyfta fram lättlästa böcker med tekniskt djup.   

---
## User stories för vyn "Katalog"

---

### US_1: Välkomnstmeddelande
**Som en** användare, **vill jag** se ett välkomstmeddelande när jag klickar på knappen "Katalog", **så att** jag känner mig välkommen och förstår syftet med sidan.

#### US_2: Visa boklista   
**Som en** användare, **vill jag** se en lista en katalog/lista med böcker, **så att** kan få en överblick över tillgängliga böcker.


#### US_3: Markera favoritbok
**Som en** användare, **vill jag** kunna markera en bok som favorit med ett hjärtikon, **så att** jag kan snabbt välja mina favoritböcker.

#### US_4: Rader ändrar färg vid mouseover
**Som en** användare, **vill jag** att raderna i boklistan ändrar färg när jag för musen över dem, **så att** jag kan se vilken rad jag är på.

## User stories för vyn "Lägg till bok"

---

### US_1: Se en formulär för att lägga till en ny bok
**Som en** användare, **vill jag** se ett formulär med fält för titel och författare när jag klickar på knappen "Lägg till bok", **så att** jag kan fylla i för att lägga till en bok i katalogen efter att jag klickar på formulärens knapp "Lägg till ny bok".

### US_2: Lägg till en bok som finns i katalogen
**Som en** användare, **vill jag** få ett felmeddelande när jag försöker lägga till en bok som redan finns i katalogen, **så att** jag inte råkar lägga till dubbletter.

### US_3: Felmeddelande vid tomma fält
**Som en** användare, **vill jag** få ett felmeddelande när jag försöker lägga till en bok utan att fylla i både titel och författare fälten, **så att** jag inte råkar lägga till ofullständig information i katalogen.

### US_4: Formuläret återställs/rensas efter att en bok har lagts till
**Som en** användare, **vill jag** att formuläret återställs/rensas efter att jag har lagt till en bok, **så att** jag kan lägga till en ny bok utan att behöva rensa fälten manuellt.

## User stories för vyn "Mina böcker"

---

### US_1: Visa lista på favoritböcker
**Som en** användare, **vill jag** se ett välkomstmeddelande när jag klickar på knappen "Mina böcker", **så att** jag kan se en sammanställd ordnad lista på de böckerna jag markerade som favoriter i "Katalog" vyn.

### US_2: Tom sida när inga favoritböcker har markerats
**Som en** användare, **vill jag** se en tom sida om jag inte har några favoritböcker, **så att** jag kan se att det stämmer med att jag inte valde några böcker.

### US_3: Listan uppdateras när jag markerar/avmarkerar en bok som favorit
**Som en** användare, **vill jag** att listan över favoritböcker uppdateras i realtid när jag markerar eller avmarkerar en bok som favorit i "Katalog" vyn, **så att** jag kan se mina favoritböcker utan att behöva uppdatera sidan manuellt.

## User stories för vyn "Statistik"

---

### US_1: Visa statistik över antal böcker i boklistan
**Som en** användare, **vill jag** se en statistik över det totala antalet böcker i boklistan när jag klickar på knappen "Statistik", **så att** jag kan få en överblick över hur många böcker som finns i katalogen.

### US_2: Visa statistik över antal favoritböcker
**Som en** användare, **vill jag** se en statistik över det totala antalet favoritböcker när jag klickar på knappen "Statistik", **så att** jag kan få en överblick över hur många böcker användarna har markerat som favoriter.
